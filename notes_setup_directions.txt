🟢 Future State (Claude Automation)

Three Python modules doing the heavy lifting:

┌──────────────────┬────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│       File       │                                                    Role                                                    │
├──────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ dbt_analyzer.py  │ Scans your SQL + YAML → extracts columns, CTEs, CASE branches, business rules from comments                │
├──────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ claude_client.py │ Calls Claude claude-sonnet-4-6 with prompt caching (system prompt cached = 90% discount on repeated calls) │
├──────────────────┼────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│ test_writer.py   │ Writes 3 artefacts per model + an ROI audit report                                                         │
└──────────────────┴────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

Run it:
cd future_state/auto_test_generator
pip install -r requirements.txt
cp .env.example .env   # add your ANTHROPIC_API_KEY

# Single model (~30 seconds)
python main.py --model stg_orders

# Whole project + ROI summary
python main.py --all
